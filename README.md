# Appu Chat Center Kalaburgi Website

This project is a simple responsive website created for a local street food business called Appu Chat Center located in Kalaburgi.

The website includes:

• Landing page  
• Popular food menu section  
• Business information  
• Contact details  
• Google Maps location integration  

Technologies Used:

- HTML
- CSS
- JavaScript

This project is part of my frontend development practice and portfolio.
